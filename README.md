MSc Artificial Intelligence Student Portal

This repository contains coursework pages and supporting files prepared by Alex Tsaggas for the MSc in Artificial Intelligence programme delivered by Aegean College in partnership with the University of Essex.

Available Module

AI117 — Introduction to A.I. and Machine Learning

The module page includes Altair AI Studio process files covering Decision Trees, Data Analysis and Linear Regression, K-Means Clustering, Neural Networks and Imbalanced Dataset Evaluation.

Open the published student portal to access the module resources and download the available process files.
